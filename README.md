## Overview

TripBuddy is a beautifully designed Android app that helps you plan your next adventure, track your budget, and capture timeless memories. Recently modernized with Google's latest **Material 3 Design Principles**, it offers an incredibly fluid, edge-to-edge experience tailored to your device's personal aesthetics using Dynamic Colors.

## Features

- **Dynamic Color (Material You):** The entire app automatically adapts its color scheme based on your system wallpaper.
- **Modern Navigation:** Intuitive bottom navigation powered by Jetpack Navigation components for a seamless single-activity flow.
- **Trip Planning & Budgeting:** Easily calculate trip costs, add custom expenses, and generate real-time budget summaries.
- **Memory Gallery:** Save your favorite moments and explore them in a beautiful, edge-to-edge adaptive grid layout.
- **Premium Micro-Interactions:** Enjoy immersive shared-element transitions (like images expanding elegantly) and custom haptic feedback on key architectural moments.

## Tech Stack

- **Platform:** Android (Java)
- **UI Architecture:** Material 3, ViewBinding, CoordinatorLayouts, MaterialCardViews
- **Navigation:** Jetpack Navigation Component
- **Local Storage:** Room Database, SharedPreferences
- **Build System:** Gradle (AGP 8.11+)

## Installation & Build

To run TripBuddy locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/malawaubi-ship-it/Trip-Buddy.git
   ```
2. Open the project directory (`Original_Project/TripBuddy4`) in **Android Studio**.
3. *Note:* Ensure your IDE is configured to use **Java JDK 17** (required for the Android Gradle Plugin configuration).
4. Run Gradle Sync and click **Run** (`Shift + F10`) to deploy to an emulator or physical device.
