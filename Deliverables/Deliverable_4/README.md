Deliverable 4 — Multimedia & Interactivity

4.1 Background music
- MemoriesActivity allows associating audio tracks with memory posts.
- Integrated MediaPlayer with controls for play, pause, and stop.
- Lifecycle managed: music stops when user exits the screen.

4.2 UI animations
- Basic animations implemented through fade-in transitions and screen changes.
- Animated buttons and visual feedback improve interactivity.

4.3 Dynamic photo grid
- GalleryActivity displays user-uploaded trip photos using RecyclerView with GridLayoutManager.
- Ensures responsive grid layout for photo collections.

4.4 Tap interactions
- When a gallery photo is tapped, FullScreenPhotoActivity opens the image in detail view.
- Supports playing associated background music and animations during viewing.
